# Дизайн: Модуль "Анализ новостей" для Alfa-Bank Sales Bot

**Дата:** 2026-06-23  
**Статус:** Approved  

---

## 1. Цель

Добавить в Telegram-бот раздел "Анализ новостей", позволяющий пользователю получить AI-анализ влияния любой финансовой новости на конкретные инвестиционные продукты Альфа-Банка (ОПИФ, ОМС, Стратегии автоследования, общий рынок РФ). Поддерживается ручной ввод новости и автоматический парсинг за последние 48 часов.

## 3. Техническая архитектура

### Новые файлы

| Файл | Назначение |
|---|---|
| `states/news_analysis.py` | FSM-состояния модуля |
| `handlers/news_analysis.py` | Все callback- и text-handler'ы |
| `services/news_fetcher.py` | Async-парсинг новостей с двух сайтов |
| `prompts/news_prompts.py` | AI-промпты с составом каждого продукта |

### Изменения в существующих файлах

| Файл | Изменение |
|---|---|
| `handlers/start.py` | Новая кнопка + обновлённый текст `/start` |
| `bot.py` | Подключить `news_analysis.router` |
| `services/claude.py` | Новая функция `analyze_news_impact()` |

### FSM-состояния (`states/news_analysis.py`)

```python
class NewsAnalysis(StatesGroup):
    choosing_category   # верхний уровень: ОПИФ / ОМС / Стратегии / Общее
    choosing_product    # конкретный продукт внутри категории
    choosing_input_mode # ручной ввод или авто-48ч
    waiting_news        # ожидаем текстовое сообщение с новостью
```

### Сервис парсинга (`services/news_fetcher.py`)

- `aiohttp` для async HTTP-запросов
- Источники:
  - `https://alfabank.ru/alfa-investor/`
  - `https://bcs-express.ru/category`
- Фильтр: только новости за последние 48 часов
- Лимит: максимум 10 новостей (5 с каждого источника) во избежание перегрузки контекста Claude
- Возвращает: `list[dict]` с полями `title`, `text`, `date`, `source`
- При недоступности одного источника — используется второй
- При недоступности обоих — возвращает пустой список

### AI-анализ (`services/claude.py`)

Новая функция:
```python
async def analyze_news_impact(
    news_text: str,
    product_id: str,
    product_context: str,
) -> str:
```

- `model`: `claude-sonnet-4-6`
- `max_tokens`: 400
- Промпт: состав продукта + новость + инструкция на лаконичный ответ
- Включает индикатор влияния: 📈 Позитивное / 📉 Негативное / ➡️ Нейтральное

## 4. Состав продуктов для AI-промптов

### ОПИФ

**Облигации Плюс**
- 50.3% ОФЗ (Минфин РФ), корп. облигации: ГТЛК 7.6%, РЖД 5.1%, Атомэнергопром 4.2%, РусГидро 3.8%, Банк ВТБ 1.7%
- Отрасли: госбумаги 50%, банки 11%, электроэнергетика 10%, промышленность 9%
- Чувствителен к: ставке ЦБ, кредитному качеству корпоратов, госфинансам

**Облигации с выплатой дохода**
- 25.5% ОФЗ, банки/финансы 28.4%, промышленность 8.6%, энергетика 6.8%
- Топ: Минфин РФ 24.6%, ГТЛК 7.9%, Авто Финанс Банк 5.6%
- Ежеквартальные выплаты купонного дохода
- Чувствителен к: ставке ЦБ, кредитному риску банков

**Сбалансированный с выплатой дохода**
- ~71% акции, ~16% ОФЗ, ~13% корп. облигации
- Топ акции: Сбербанк 7.8%, HeadHunter 7.2%, Яндекс 6.0%, Т-Технологии 5.6%
- ОФЗ: Минфин РФ 16.3%
- Ежеквартальные выплаты
- Чувствителен к: рынку акций РФ, геополитике, ставке ЦБ

**Управляемые акции с выплатой дохода** (БПИФ)
- ~100% акции: нефтегаз 26%, финансы 21.5%, IT 16.5%, потребительский 13.4%
- Топ: Сбербанк 9.6%, Ozon 8.3%, Яндекс 8.1%, Новатэк 7.9%, Лукойл 6.2%
- Ежеквартальные выплаты дивидендов
- Чувствителен к: ценам на нефть, санкциям, корп. событиям, геополитике

**Денежный рынок** (БПИФ)
- 100% РЕПО с ЦК — короткие рублёвые инструменты
- Доходность ≈ ключевая ставка ЦБ минус комиссия
- Минимальный риск, высокая ликвидность
- Чувствителен к: решениям ЦБ по ключевой ставке (прямая зависимость)

**Управляемые облигации** (БПИФ)
- 29% ОФЗ, корп. облигации 1-2 эшелона
- Отрасли: госбумаги 29%, банки 19.7%, энергетика 17%, промышленность 13.5%
- Топ: Минфин 29%, ГТЛК 9.3%, Атомэнергопром 9.2%, Сэтл Групп 6.8%, РЖД 6.6%
- Чувствителен к: ставке ЦБ, кредитному качеству 1-2 эшелона

### ОМС (Обезличенные металлические счета)

| Металл | Ключевые факторы влияния |
|---|---|
| Золото | USD/RUB, геополитика, ставка ФРС, инфляция, спрос ЦБ мира |
| Серебро | Промышленный спрос, солнечная энергетика, USD/RUB |
| Платина | Автопром (катализаторы), спрос из Европы, дефицит/профицит рынка |
| Палладий | Автопром (бензиновые двигатели), Россия — крупнейший производитель |

### Стратегии автоследования

**Тихая Гавань**
- 40% Денежный рынок
- 40% Управляемые облигации
- 10% Флоатеры (переменный купон)
- 10% ОФЗ
- Характер: консервативная, защитная. Чувствительна к ставке ЦБ.

**Навигатор фондов**
- 50%+ Денежный рынок
- 30% Фонд облигаций с плавающей ставкой (флоатеры)
- 10% Облигации с фиксированным доходом
- 3% Акции
- 2% Золото
- Характер: ультраконсервативная. Максимально чувствительна к ставке ЦБ.

**Вечный портфель**
- 30% Денежный рынок
- 30% Управляемые российские акции
- 30% Управляемые облигации
- 10% Золото
- Характер: сбалансированная. Чувствительна к рынку акций, ставке ЦБ, геополитике.

## 6. Edge Cases

| Ситуация | Поведение |
|---|---|
| Пользователь отправил голос вместо текста | "Пожалуйста, отправьте текст новости" |
| Парсер не нашёл новостей за 48ч | "Свежих новостей не найдено. Вставьте новость вручную." |
| Оба источника недоступны | Сообщаем об ошибке, предлагаем ручной ввод |
| Новость длиннее 3000 символов | Берём первые 3000 символов, предупреждаем пользователя |
| `/cancel` в состоянии waiting_news | Возврат в главное меню, очистка состояния |
| В состоянии waiting_news — нажата кнопка | Обрабатываем кнопку, не ломаем FSM |
